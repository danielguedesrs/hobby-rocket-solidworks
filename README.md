# Hobby Rocket - SolidWorks Project

Projeto de modelagem 3D de um foguete experimental desenvolvido no SolidWorks.

## Objetivos
- Desenvolver habilidades em CAD
- Aplicar conceitos de foguetemodelismo
- Simular estrutura e estabilidade

## Componentes
- Nose cone
- Corpo principal
- Aletas
- Adaptador de motor

## Ferramentas utilizadas
- SolidWorks
- SolidWorks Simulation
